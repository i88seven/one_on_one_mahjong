# one_on_one_mahjong

Mahjong game we can fight one-on-one

## Getting Started

Made by Flutter. [online documentation](https://flutter.dev/docs)

```bash
$ cp .env.example .env
$ flutter pub get
$ flutter run
```

## Release

Create `.env` as production. Refer [AdMob](https://apps.admob.com/v2/home).
